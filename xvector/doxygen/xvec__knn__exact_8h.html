Detailed Description

Exact KNN (K-Nearest Neighbors) brute-force search interface.

This header provides the API for loading and searching datasets using exact brute-force nearest neighbor search. KNN computes distances between every query vector and every target vector, providing exact results.

The parallel execution mode is automatically selected:

  • When queryCount >= taskCount: queries are partitioned across workers
  • When queryCount < taskCount: the dataset is partitioned across workers

Performs a brute-force K-nearest neighbors search.

Computes exact distances between every query vector and every target vector in the index, returning the top-K nearest neighbors for each query.

The parallel execution mode is automatically selected based on the number of queries and the device task count.

Parameters
[in]contextExecution context for device operations
[in]indexHandleHandle to the KNN index containing target vectors
[in]searchQuerySearch query parameters (queries, topK)
[out]resultsBufferHandleBuffer to receive search results. Must be pre-allocated with size: queryVectorCount * topK * sizeof(xvecKnnExactSearchResultFloat32)
[out]outTimingPointer to receive timing information (can be NULL)
Returns
XVEC_SUCCESS on success, error code otherwise
