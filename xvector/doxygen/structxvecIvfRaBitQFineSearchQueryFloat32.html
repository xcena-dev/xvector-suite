Detailed Description

Search query parameters for IVF-RaBitQ fine search only.

This structure is used when the coarse search has already been performed externally (e.g., by FAISS) and only the fine search needs to be executed on XVector. Unlike xvecIvfRaBitQSearchQueryFloat32, this structure takes pre-computed nearest cluster IDs instead of numProbes.
