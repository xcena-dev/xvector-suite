Detailed Description

C++ configuration management API for the XVECTOR library.

This header provides a C++ API for centralized configuration management. Configuration values are loaded from multiple sources with the following priority order (highest to lowest):

  1. Environment variables
  2. Default values

Supported environment variables:

  • XVECTOR_MUBIN_PATH: Path to mubin file
  • DEVICEID: Device ID (uint32)
  • NUMSUB: Number of subs (uint32)
  • TASKCOUNT: Task count (uint32)
  • LOGHOST: Host log level (0=NONE, 1=ERROR, 2=WARN, 3=INFO, 4=DEBUG, 5=TRACE; also accepts level names and true/false)
  • LOGDEVICE: Device log level (0=NONE, 1=ERROR, 2=WARN, 3=INFO, 4=DEBUG, 5=TRACE; also accepts level names and true/false)

Example usage:

#include <xvector_cpp/xvec_config.hpp>
auto& config = xvec::ConfigManager::instance();
uint32_t deviceId = config.getDeviceId();
auto mubinPath = config.tryGetMubinPath(); // std::optional<std::string>
