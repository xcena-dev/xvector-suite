Detailed Description

Log sink registration API for the XVector library.

This header provides the public API for registering custom log sink callbacks to receive log messages from the library.

Sets the minimum log level for runtime filtering.

Only log messages with severity equal to or higher than the specified level will be passed to the log sink. This provides runtime control over log verbosity.
